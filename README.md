## Running the project locally

Running locally requires having Node installed.

```sh
# Step 1: Install dependencies
npm install

# Step 2: Run the local dev server
npm run dev
```

Project will be served on http://localhost:5155/ (or next available port if necessary).
Note: No environment variables or authentication needed.

### Compile and Minify for Production

```sh
npm run build
```

### Lint & Format

```sh
npm run lint
npm run format
```
